# Task Manager App (React Native + Express + SQLite)

A full-stack **Task Manager** application built using **React Native (Expo)** for the frontend and **Node.js (Express + SQLite)** for the backend.

This project was developed as part of a technical assignment for a React Native Developer role.

---

## Tech Stack

**Frontend**
- React Native (Expo)
- TypeScript / JavaScript
- Expo Router
- Fetch API for REST communication

**Backend**
- Node.js
- Express.js
- SQLite (via `better-sqlite3`)

---

## Features

✅ Add new tasks with title & description  
✅ View all tasks in a simple clean interface  
✅ Filter tasks by status — All, Pending, Completed  
✅ Search tasks by title  
✅ Mark tasks as completed  
✅ Data persistence using SQLite  
✅ Works seamlessly via LAN or public tunnel (LocalTunnel / Ngrok)
